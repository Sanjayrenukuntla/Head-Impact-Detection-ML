# Machine Learning-Based Head Impact Detection

## Overview
A predictive analytics web application designed to detect and classify head impact locations based on sensor data. This project utilizes machine learning algorithms to improve safety monitoring in high-risk environments.

## Tech Stack
* **Language:** Python
* **Algorithms:** XGBoost, Scikit-Learn
* **Data Processing:** Pandas, NumPy
* **Web Interface:** Flask (for model deployment)

## Key Features
* Achieved **90% accuracy** in impact location prediction.
* Optimized data pipeline using Pandas for handling large datasets.
* Real-time prediction interface using a web-based dashboard.
